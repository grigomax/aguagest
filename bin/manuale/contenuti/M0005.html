Requisiti minimi

I requisiti sono minimi in quanto agua può funzionare in varie modalità, server, e Client-Server.

Sulla macchina base ci deve essere installato e funzionante, un server http, es. Apache, il PHP e mysql per il database server. Per i requisiti minimi, bisogna rivolgersi al vari programmi nei vari siti, comunque non necessitano di una Ferrari, a meno che non usate Windows. sul client almeno un BROWSER internet per navigare in agua.

La velocità di esecuzione dipende da molti fattori, Potenza server, Client , Rete.


Per installazioni sotto windows consiglio easyphp scaricabile gratuitamente su www.easyphp.org, per quanto riguarda le installazioni MAC consiglio di installare il sistema MAMP.

Per le installazioni RPM based, esiste già il pacchetto RPM con la installazione automatica delle dipendenze.

Il programma è nativo per:

LINUX > 2.6

MAGEIA > 2

APACHE SERVER > 2.0

PHP > 5.0.0 con librerie PDO, ZIP, FTP

MYSQL SERVER (mariadb)

OPERA BROWSER > 12.00
