La prima connessione ed installazione

La prima Connessione

La Prima connessione va effettuata aprendo un browser e inserendo l\'indirizzo base del programma..

 

http://localhost/agua/index.php

 

Qui appare la prima schermata.. ed inseriamo come indentificativo..

 

Nome utente : admin

Password : admin

 

Ora, il programma effettua una serie di controlli sulla possibilità di scrivere ed aggiornare le cartelle ed i file base di personalizzazione.

 

Effettuati questi controlli, caratterizzati con la buona riuscita [OK] scritta in verde, è possibile proseguire, seguendo il link in basso, questo porterà ad l\'inserimento dei dati basilari di connessione al database ed di impostazione del nome della società.

 

Parametri di installazione server HTTP

Nome Host:

Indirizzo di dove risiedono i file del programma generalmente \"localhost\"

Percorso Assoluto:

Indirizzo fisico della cartella all\'interno del server dopo l\'indirizzo locale generalmente \"/agua\"

 

Parametri DATABASE

Nome host server mysql:

Indirizzo internet di dove risiede il server mysql di solito \"localhost\"

Nome utente mysql

Nome di connessione per accedere ai dati del database \"root\"

Password mysql

Password impostata nel database per poter accedere ai dati ****

Nome archivio da assegnare al programma.. generalmente agua

Intestazione Attività

Nome azienda da impostare come base per tutte le anagrafiche, poi verrà in un secondo momento aggiornata con tutti i dati aziendali.

Una volta completato l\'inserimento dei dati proseguiamo al passo successivo..

Ora se la creazione è andata a buon fine il programma ha settato i parametri base..

Ora creiamo le tabelle e l\'inserimento dei dati basilari per poter inziare ad essere operativi subito con il programma..

Ecco ora abbiamo quasi finito, con il prossimo passo l\'installazione è completata e si potrà accedere al programma per poter andare nella sezione parametri aziendale..
