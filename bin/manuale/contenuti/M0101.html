Capitolo 1 - Installazione

Installazione

Per la prima installazione bisogna scaricare il programma dal sito internet.

http://aguagest.sourceforge.net

Qui bisogna scegliere quale file scaricare, esistono due tipi, uno è quello dedicato a linux in formato RPM e l\'altro è quello generico per le altre distribuzioni, per windows ed apple in formato ZIP.

 

Per i formati RPM

Per chi ha la fortuna di aver linux ed il formato rpm, il pacchetto auto-installante verifica le dipendenze di agua gest ed in caso li installa, se il server apache mysql e php-pdo non sono presenti, vengono installate di conseguenza. Può essere che il gestore dei pacchetti avvisi che la firma non è corretta, ma potete accettare tranquillamente. 

Il programma si installa di default nella directory /var/www/html/agua 

Verrà aggiunto tramite urpmi una nuova fonte di gestione dei pacchetti in modo che il programma si tenga aggiornato automaticamente.

Una volta finito di installarsi, basta aprire il browser all\'indirizzo http://localhost/agua/index.php e seguire le indicazioni successice.

 

Per i formati ZIP ( windows, apple, altri OS)

Per far funzionare il programma si necessita di un server apache, mysql e php con pdo già installati e perfettamente funzionanti.

Una volta scaricato il programma, va creata una cartella agua all\'interno della vostro server html all\'interno della cartella indice, essa e da cercare in quanto ogni sistema operativo o programma server la crea con nomi diversi in posti diversi all\'interno del vostro server(pc).

Una volta scompresso il programma, 

basta aprire il browser all\'indirizzo http://localhost/agua/index.php e seguire le indicazioni successive.
