Gestionale AGUA GEST
Gestionale Aziendale Web-Based
Basato su piattaforma php - mysql - apache
Scritto su Browser Chromium e derivati (webkit)
by GRIGOMAX

La versione RPM è un file autoinstallante per Linux adattato per Mageia,
ma dovrebbe funzionare anche per redhat, fedora e mandriva.

La Versione ZIP è per tutti gli altri sistemi operativi.

Necessita di un server funzionante.
